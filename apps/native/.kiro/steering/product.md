# Product Overview

Outfit Planner is a cross-platform mobile application built with Expo and React Native. The app provides user authentication and a foundation for outfit planning features.

## Core Features

- User authentication (sign in/sign up) via Better Auth
- Light/dark theme support with system preference detection
- Drawer and tab-based navigation
- Modal screens for dialogs and confirmations

## Target Platforms

- iOS
- Android
- Web (via Expo)
